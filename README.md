# ZESTY 🍋

A modern fullstack restaurant web app built with:

- React (Vite)
- Tailwind CSS
- Node.js + Express
- MongoDB

## Features
- Interactive menu (flipbook + mobile view)
- Image gallery
- Responsive design
- Backend API with database

## Tech Stack
Frontend: React + Vite + Tailwind  
Backend: Node.js + Express  
Database: MongoDB  

## Status
Work in progress / Portfolio project
